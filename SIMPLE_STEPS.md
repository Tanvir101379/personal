# 🚀 DEPLOY IN 15 MINUTES - Simple Steps

Follow these exact steps to launch your portfolio live.

## STEP 1: Prepare Avatar Image (5 min)

### Get Your Photo
- Use your LinkedIn profile photo
- Or any professional headshot

### Resize to 300x300px
1. Go to https://www.befunky.com/create/crop-photo/
2. Upload your photo
3. Crop to square (300x300px)
4. Download as JPG

### Compress & Save
1. Go to https://tinypng.com/
2. Upload your image
3. Download compressed version
4. Save as: `avatar.jpg` in your outputs folder

4. Click repository
5. Click "Deploy site"

### Wait

- Deployment completes
- Shows green checkmark

## 🆘 If Something Goes Wrong

### "Files won't upload"
- Make sure repository is PUBLIC
- Try uploading 1-2 files at a time

### "Avatar doesn't show"
- File must be named `avatar.jpg` (lowercase)
- Size must be 300x300px
- Should be JPG format

### "Site shows 404"
- Wait 1-2 minutes (still building)
- Hard refresh (Ctrl+Shift+R)
- Check netlify.toml is uploaded

### "Design looks broken"
- Hard refresh (Ctrl+Shift+R)
- Clear browser cache
- Try different browser

### "Can't find deploy button"
- Make sure you're logged into Netlify
- Make sure GitHub is connected
- Try logout/login again
